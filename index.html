Recent Updates

20 Jan 2025

Second PhD Manuscript in Progress

Currently finalizing the second manuscript of my PhD thesis on "Capabilities, Limitations, and Optimization Strategies of using LDI-FT-ICR MS for DOM Corona Characterization on Nanoparticles". This comprehensive study explores the analytical potential and challenges of Laser Desorption/Ionization Fourier Transform Ion Cyclotron Resonance Mass Spectrometry for characterizing dissolved organic matter corona formation on nanoparticles.

PhD Research Manuscript LDI-FT-ICR MS ES&T
15 Sep 2025

Published Code: NP_SRFA_Ratio Analysis Pipeline

Released the complete R-based pipeline for analyzing molecular formula patterns in Suwannee River Fulvic Acid (SRFA) samples and comparing them across gold nanoparticles (AuNP) and P25 TiO₂ nanoparticles. This reproducible computational framework includes data preprocessing, group-wise aggregation, intensity-weighted averages, and publication-ready visualizations including Van Krevelen diagrams.

Code Release R Pipeline Nanoparticles ES&T
View Repository
15 Sep 2025

Published Code: DOM-Drivers Analysis

Released the computational code for the DOM composition analysis published in Water Research. This repository contains the machine learning models and analytical workflows used to identify environmental drivers of dissolved organic matter composition across central European aquatic systems using FT-ICR MS data.
